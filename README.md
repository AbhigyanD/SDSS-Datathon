# SDSS-Datathon — Airline Tickets Case

Analysis of U.S. domestic airfare markets using DOT data (2021–2025 Q2): when, where, and why airfares differ, and how city-level characteristics influence route-level prices.

## Setup

```bash
pip install -r requirements.txt
```

## Run

1. Open `airline_analysis.ipynb` in Jupyter (or VS Code / Cursor).
2. Run all cells (Kernel → Run All). The notebook loads `airline_ticket_dataset.csv` from this directory.
3. If your dataset uses different column names, adjust the mapping in **Section 2** and re-run.

## Deliverables

- **Code:** This repo (README + `airline_analysis.ipynb`).
- **Slides / Video:** Per Datathon instructions (max 15 slides, 6 min video).
